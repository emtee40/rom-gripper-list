Emulator Game List

Steps
  1. Select a game system

    The game list of this game type will shown.
  2. Select a game

    The game details page will show. The screenshots will be shown if available.
  3. Search on Internet
  4. Choose one link
    Most of the time, you can download with www.coolrom.com. So click the link from coolrom.com
  5. Download
    For legal reasons, please ensure that you own the physical game that you are downloading the ROM for.
    Click the "Download Now" button to download
  6. Download an emulator app of the game type
    On game list page, click "Look for an emulator app"

    Select a store you like

    Choose a proper emulator app to install. For example, you can install John NES Lite (NES Emulator) if you want to play an NES game.
  7. Enjoy it and bring back your childhood
    Play the game with your emulator app.
