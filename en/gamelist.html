Steps
  1. Select a game type

    The game list of this game type will shown.
  2. Select a game

    The game details page will show. The screenshots will be shown if available.
  3. Copy ROM file name

    The standard ROM file name is copied to the system clipboard. If you don't use this name when you download, the Play button above will not be enabled.
  4. Search on Internet

    It is recommended to use Dolphin Browser.
  5. Choose one link
    Most of the time, you can download with www.coolrom.com. So click the link from coolrom.com
  6. Download
    Click the "Download Now" button to download

    Paste the ROM file name.

    Change the save path to the one shown in Step 3 on details page. If it doesn't exist, create it.

    Click the "Save" button to download the ROM.
  7. Download an emulator app of the game type
    On game list page, click "Look for an emulator app"

    Select a store you like

    Choose a proper emulator app to install. For example, you can install John NES Lite (NES Emulator) if you want to play an NES game.
  8. Enjoy it and bring back your childhood
    Open the ROM file downloaded to play the game
